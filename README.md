# New Schedule Contact Component

New Schedule Contact component for SmartTracker App

## Install

```sh
meteor npm install
```
## Methods

- agencyData
- allCarriers
- queueById

## Use
 ```sh
 <NewScheduleContact 
  ...
  agencyData={agencyData}
  allCarriers={allCarriers}
  queue={queueById}
 />
 ```
